# Project 3 - Database Integration

## Description
A Node.js, Express.js and MongoDB CRUD API for managing student records.

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Dotenv

## Features

- Create Student
- Read Students
- Update Student
- Delete Student

## API Endpoints

### Get All Students

GET /api/students

### Create Student

POST /api/students

### Update Student

PUT /api/students/:id

### Delete Student

DELETE /api/students/:id

## Installation

```bash
npm install
```

## Run Project

```bash
node server.js
```

## Server

```txt
http://localhost:3000
```
